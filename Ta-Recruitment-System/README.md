# TA Recruitment System \- README

Developer: group4

# Project Introduction

This project is a TA \(Teaching Assistant\) Recruitment Management System\. It realizes the online management of the entire process of TA position release, application and review, supports dual\-role operations for TAs and teachers, and has concise and practical functions to improve the efficiency of recruitment management\.

# Current Version Completion

The core functions have been developed\. The interface is concise and smooth, supporting the normal use of both TA and teacher roles\. There are no core function abnormalities, which can meet the basic TA recruitment business needs\.

# Executable Functions \(TA Role\)

- Login/Logout: Enter the username and password to log in; the complete personal data \(including resume\) can be normally loaded after re\-login\.

- TA Dashboard: Quickly jump to the position list, application records, and resume upload pages\.

- Position Viewing and Application: View available positions by page, view position details, and apply for positions \(duplicate applications are avoided with clear prompts\)\.

- Application Records: View all personal application records, distinguish between Pending/Approved/Rejected statuses, and view application details and refresh status\.

- Resume Upload: Upload and replace personal resumes, and view the resume upload status and file name\.

# Executable Functions \(Teacher Role\)

- Login/Logout: Teachers log in with exclusive accounts and enter the management page after completing identity verification\.

- Position Management: Release, edit and delete TA positions, and set position details \(workload, skill requirements, etc\.\)\.

- Application Review: View all TA position applications, review applications \(approve/reject\), and fill in review comments\.

- Data Viewing: View published positions and all application records to keep track of the recruitment progress in real time\.
